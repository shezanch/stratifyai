# StratifyAI

An AI-powered content strategy agent that researches trending topics 
and generates platform-specific content calendars.

## What it does
- Analyzes any website and generates social media post ideas
- Researches live Google News trends for any niche
- Generates a multi-day content calendar with captions and hashtags
- Supports Instagram, TikTok, and LinkedIn with platform-specific tone

## Built with
- Python
- Gemini API
- BeautifulSoup
- Google News RSS
- Streamlit
